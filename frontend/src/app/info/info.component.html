Information

To get a face detection resistant image you have to consider a couple of important points.

The results of the different methods very a lot dependent of the image size and color. Before downloading the new image it is highly recommended to test the resistancy against all available cascades. Each cascade is a model trained with different input values. Additionally you shoud try to test the Sensitivity at "1" and the Scale-Factor at the lowest level possible. That causes the algorithm to detect way more False-Positives, but we want to be sure that your face will not be detected under most if not all circumstances. Thus you can be sure that your image will really be resistant against the Viola-Jones Framework. If your face gets detected through only one of the Cascades, please try a different convertion method. This method will not protect your image. More details and the used cascades can be fund here:
OpenCV documentation or Cascades

If there are any questions left, check out our FAQ site.

The size of your image should not exceed 5MB !

Image convertion takes ~ 2 - 20 sec.

Disclaimer:

There is no 100% guarantee that the converted image is face detection resistant against all kind of face detection frameworks!

Description of the available methods:

Method Description
Hide half Hides the lower part of the image behind a black rectangle with an alpha of 80%
Black grid Every 2nd pixel of the image will be black
White grid Every 2nd pixel of the image will be white
Edges Only detected edges of the image will be displayed
HSV Color Changes the color scheme of the image to HSV
Invert color Inverts every pixel of the image
Invert gray color Converts the image into a grayscale image and inverts every pixel of it
Cartoon Cartoonizes the image
Color highlighting Converts a threshold of white color to black and highlights it
Blue filter Shows only the blue channel of the image
